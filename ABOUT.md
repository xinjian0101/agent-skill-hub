# About Agent Skill Hub

## Suggested GitHub About

Curated catalog of role-based development workflow templates with examples, limitations, versions, and source attribution.

## Suggested topics

`developer-tools`, `workflow-templates`, `catalog`, `productivity`, `documentation`, `open-source`

## Purpose

The repository will organize reusable role-based workflow templates in a consistent, reviewable format.

## Intended users

Developers, product teams, maintainers, educators, and independent builders.

## Current status

Foundation phase. The catalog schema and acceptance rules are being defined.

## Maintenance

Entries should be versioned, reviewed, and accompanied by examples. Catalog counts must reflect committed content only.
