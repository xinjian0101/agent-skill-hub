# Agent Skill Hub

A curated catalog of reusable role-based development workflow templates.

Current status: foundation phase. No stable catalog release is available.

## Planned catalog

- Product planning templates
- User-interface review templates
- Frontend and backend workflow templates
- Testing and documentation templates
- Data-analysis templates
- Project-maintenance templates

## Quality rules

Every catalog entry should include a clear purpose, expected inputs, outputs, limitations, examples, version information, and source attribution.

## Roadmap

1. Define the catalog schema.
2. Publish reference entries.
3. Add validation.
4. Add versioned releases.
5. Expand the reviewed catalog.

## Maintenance

Catalog claims will reflect committed and reviewed entries only.
